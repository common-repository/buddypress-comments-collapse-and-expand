Adds Show/Hide comments for activity stream items in a buddypress site.

== Description ==

Add a show comments replacement for comments that exceed(x) you can specify (x) in the admin panel.

== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `BuddyPress Comments Collapse Expand` and find this plugin in the results
3. Click 'Install'
4. Once installed, activate via the 'Plugins -> Installed' page
5. Click Bp Comments link under the BuddyPress admin menu
6. Specify the options you want

= Manual Installation =

1. Upload `buddypress-comments-collapse-expand` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
5. Click Bp Comments link under the BuddyPress admin menu
6. Specify the options you want
